# Stamps Steel Buildings Website

Preliminary static website for Stamps Steel Buildings — a new pre-engineered metal building company based in Bethpage, Middle Tennessee.

## Live Site
Once GitHub Pages is enabled: https://SpiderForce-Star.github.io/Stamps-Steel/

## Quick Setup for Images
1. In this repo, create a folder named `images`.
2. Upload / rename:
   - The white/black wood-soffit building (CNJKw.jpg) → `images/hero.jpg`
   - Alabama Auto Customs building → `images/commercial.jpg`
3. (Optional) Add more from your rendered/mockup images as `images/project3.jpg` etc. and update the HTML srcs.
4. The site already has Unsplash fallbacks so it looks professional immediately.

## Enable GitHub Pages
1. Go to Settings → Pages
2. Source: Deploy from a branch
3. Branch: main, folder: / (root)
4. Save. Site will be live in 1-2 minutes.

## Customize
- Update phone, email, and form action in the Contact section of `index.html`
- Edit About text, services, or add more sections
- All pure HTML + Tailwind via CDN — no build step, easy to edit even as a coding novice
- Safe, free, no monthly costs
